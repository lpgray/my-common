loading.hide(); // 关闭loading

confirm面板
可以自定义reset及confirm回调函数。

API
$('#domId').confirms({
	reset : function(){
		alert('这是reset回调函数');						
	},
	confirm : function(){
		alert('这是confirm回调');
	}
});

ajax获取数据
ajax向后台请求数据，数据以特定jQuery方法，显示在指定的dom内

API

首先需要给触发事件的按钮添加一个类 a4d 一个 data-result-ctn="jquery选择字符串[(jquery方法)]" 我这里写的是div的id。 以及需要ajax访问的url data-url="url" 或者 href="url"。 默认情况下 ajax for data 特性是被激活的，你不需要写任何 javascript 代码。 

<button href="test/ajax4data.json" data-result-ctn="#a4dctn1" class="btn btn-large btn-info a4d">ajax4data(默认)</button>
<button data-url="test/ajax4data.json" data-result-ctn="#a4dctn2(append)" class="btn btn-large btn-info a4d">ajax4data(默认)</button>
